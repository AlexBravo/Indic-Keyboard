## Indic IME
AOSP keyboard ported to support more languges.

## Requirements

* Android 4.1 and above.

## Supported Languages
* Assamese
* Bengali
* Gujarati
* Hindi
* Kannadda
* Malayalam
* Marathi
* Nepali
* Oriya/Odia
* Punjabi
* Sanskrit
* Sinhalese
* Tamil
* Telugu
* Urdu
